Server example

This file is being served by package:shelf_static from the /public directory.

This app uses package:shelf_router to define several code-based handlers:

  • [GET] /api/count
  • [POST] /api/count/increment
  • [POST] /api/count/decrement

  • [POST] /api/authenticate
  • [POST] /api/logout

  • [POST] /api/user/push-notification-subscriptions
  • [DELETE] /api/user/push-notification-subscriptions
